
  # StudyLex — твой карманный репетитор английского

Telegram-бот для изучения английских слов. Легко переводит слова с английского на русский и с русского на английский, помогая пополнять словарный запас в удобном формате.

### 🚀 Функции
*   **Перевод в обе стороны:** С английского на русский и с русского на английский.
*   **Быстрый и удобный:** Просто отправь боту слово — получи перевод.

### 🤖 Как начать?
  Нажми **"Start"** или отправь команду `/start`. 
  Просто отправь любое слово на русском или английском языке.
